# InsightPrep - 기업 분석 & CS 면접 학습 플랫폼

## 프로젝트 요약

한 줄 설명: 기업 분석을 AI가 도와주고, 실시간 채팅방에서 지원자들과 정보를 교류하며, CS 면접 대비도 GPT가 피드백해주는 실전 중심의 AI 기반 면접 학습 플랫폼입니다.

기간: 2025.07 ~ 2025.09

인원: 김경호(FE, BE)
